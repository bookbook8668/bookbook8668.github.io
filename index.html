Variables

Variables are used to store data values in JavaScript. They can hold various types of data such as numbers, strings, or objects.

Functions

Functions are reusable blocks of code that perform a specific task. They can be defined and called to execute the code inside them.

Conditional Statements

Conditional statements allow the execution of different actions based on different conditions. Examples include if-else statements and switch statements.

Loops

Loops are used to repeatedly execute a block of code until a certain condition is met. Common loop types in JavaScript are for loops, while loops, and do-while loops.

Objects

Objects are complex data types that can hold properties and methods. They are used to represent real-world entities and organize related data and functionality.
